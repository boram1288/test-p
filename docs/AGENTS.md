# Docs Editing Instructions

이 디렉토리의 Markdown 문서를 수정할 때는 아래 규칙을 따른다.

## 용어 표기

- `SMMU/IOMMU`는 `S2MPU`로 표기한다.
- 카메라/AI 하드웨어는 `Camera/AI HW`, `Camera HW`, `AI HW` 용어로 표기한다.
- 가운데점(`·`)은 슬래시(`/`)로 표기한다.
- 아래 음차 표기는 영어로 표기하고, 첫 글자는 대문자로 쓴다.
  - `프레임워크` -> `Framework`
  - `워크로드` -> `Workload`
  - `미들웨어` -> `Middleware`
  - `컨텍스트` -> `Context`
  - `하이퍼바이저` -> `Hypervisor`

## 문서 정리

- `01_requirements_collection.md`의 `1.2 Stakeholder 목록` 표는 `Stakeholder`, `구분`, `역할` 컬럼만 사용한다.
- `01_requirements_collection.md`의 `3. VOS (Voice of Stakeholder) 정리` 표는 `ID`, `Stakeholder`, `VOS` 컬럼만 사용한다.
- VOS 정리 표의 `Stakeholder` 값에는 `SH-1` 같은 stakeholder 번호를 넣지 않는다.
