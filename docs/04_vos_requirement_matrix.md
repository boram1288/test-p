# VOS-Requirement Matrix

본 문서는 `01_vos_collection.md`, `02_requirements.md`, `04_architectural_drivers.md`를 기준으로 VOS와 Architectural Driver(AD) 선정 요구사항 간의 커버리지를 정리한다.

## 1. 작성 기준

- 세로축: VOS ID
- 가로축: AD로 선정된 FR, QA, CONST
- CONST는 요청 범위에 따라 `CS-01`, `CS-02`만 포함한다.
- 해당 VOS가 요구사항으로 직접 추적되는 경우 `O`로 표시한다.
